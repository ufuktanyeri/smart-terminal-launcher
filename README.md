# Smart Terminal Launcher

A VS Code extension that provides an intelligent terminal launcher with automatic terminal detection and smart command execution.

## Features

- **Terminal Detection**: Automatically detects all available terminals in VS Code
- **Smart Recommendations**: Suggests useful terminal profiles that aren't configured
- **Intelligent Execution**: Analyzes commands and runs them in the optimal terminal
- **Easy Setup**: Add recommended terminals with one click
- **Command History**: Keeps track of executed commands
- **Multi-Terminal Support**: PowerShell, CMD, Git Bash, WSL, and more

## Technology Stack

- VS Code Extension API
- TypeScript
- Node.js

## Installation

1. Install from VS Code Marketplace (Coming soon...)
2. Or install manually: Download `.vsix` file

## Usage

1. Open Command Palette (`Ctrl+Shift+P`)
2. Type "Smart Terminal Launcher"
3. Use the interface to manage and run commands
